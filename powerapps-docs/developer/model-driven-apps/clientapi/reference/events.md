# <a name="events-client-api-reference"></a>Eventos (referencia de la API de cliente)



Se producen eventos en los formularios y cuadrículas de Custom Engagement cada vez que se carga un formulario o cuadrícula, cuando se cambian datos o cuando se guardan. Ejecute el código JavaScript asociando un evento para que se ejecute cuando se produzca el evento. Más información: [Eventos de formularios y cuadrículas](../events-forms-grids.md)
