## <a name="syntax"></a>Sintaxis

`sectionObj.setVisible(bool);`

## <a name="parameter"></a>Parámetro

|Nombre|Tipo|Obligatoria|Descripción|
|--|--|--|--|
|bool|Boolean|Sí|Especifique **true** que para mostrar la sección. **false** para ocultarla.|
