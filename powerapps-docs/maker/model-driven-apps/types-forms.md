# <a name="type-of-model-driven-app-forms-in-powerapps"></a>Tipo de formularios de aplicación basados en modelos en PowerApps

 Hay diferentes tipos de formularios y cada tipo tiene una funcionalidad o un uso específicos. En la tabla siguiente se describen los tipos de formularios disponibles.  
  
|Tipo de formulario|Descripción|Más información|  
|---------------|-----------------|-----------------|  
|**Principal**|Se usa en las aplicaciones de PowerApps, la aplicación web de involucración del cliente Dynamics 365, Dynamics 365 para tablets y Dynamics 365 para Outlook.<br /><br /> Estos formularios proporcionan la interfaz de usuario principal para interactuar con datos de entidad.|[Consideraciones de diseño para formularios principales](design-considerations-main-forms.md)|  
|**Móvil**|Se usa para Dynamics 365, para las páginas de teléfonos. Este formulario simplificado se ha diseñado con el fin de utilizarse para dispositivos móviles.|[Personalización de la aplicación web](https://docs.microsoft.com/dynamics365/customer-engagement/customize/customize-phones-tablets)  |  
|**Creación rápida**|Se usa en las aplicaciones de PowerApps, la aplicación web de involucración del cliente Dynamics 365, Dynamics 365 para tablets y Dynamics 365 para Outlook.<br /><br /> En las entidades actualizadas, estos formularios proporcionan un formulario básico optimizado para crear nuevos registros.|[Creación y edición de formularios de creación rápida](create-edit-quick-view-forms.md) |  
|**Vista rápida**|Se usa en las aplicaciones de PowerApps, la aplicación web de involucración del cliente Dynamics 365, Dynamics 365 para tablets y Dynamics 365 para Outlook.<br /><br /> En las entidades actualizadas, estos formularios aparecen en el formulario principal a fin de mostrar datos adicionales para un registro al que un campo de búsqueda del formulario hace referencia.|[Creación y edición de formularios de vista rápida](create-edit-quick-view-forms.md)|  

Mientras que cada tipo de formulario tiene necesidades específicas, al trabajar con formularios se debe usar el editor de formularios. Más información: [Información general de la interfaz de usuario del editor de formulario](form-editor-user-interface-legacy.md)
