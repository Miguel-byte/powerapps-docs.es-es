## <a name="release-20590-20591"></a>Versión 2.0.590-2.0.591
1. **Mostrar datos en una galería que cambia de tamaño para ajustarse a los elementos de diferentes altos**.
   
    Agregue una galería de alto variable si tiene, por ejemplo, texto (como la descripción de un producto) cuyo largo varía notablemente. Con este enfoque se pueden mostrar las descripciones completas, sin espacios en blanco innecesarios, en una galería de vertical.
   
    Para obtener información detallada acerca de cómo mostrar datos en una galería, consulte [Show a list of items in PowerApps](add-gallery.md) (Presentación de una lista de elementos en PowerApps). Para obtener información de referencia, consulte [Gallery control in PowerApps](controls/control-gallery.md) (Control de galería en PowerApps).
2. **Determinar si las cadenas de texto comienzan por una secuencia de caracteres que se especifique**.
   
    Use la [función StartsWith](functions/function-startswith.md) para que los usuarios puedan, por ejemplo, escribir las primeras letras en un cuadro de búsqueda para filtrar una lista de elementos por el texto del principio de un campo.
3. **Determinar si las cadenas de texto se ajustan a un patrón que se especifica mediante una expresión regular**.
   
    Use la [función IsMatch](functions/function-ismatch.md) devolver un valor true/false en función de si un campo de un registro coincide, por ejemplo, con un nombre, una dirección de correo electrónico, un número de teléfono o un número del seguro social.
4. **Personalizar el orden en el que se activan los controles cuando los usuarios presionen la tecla Tab**.
   
    De forma predeterminada, las coordenadas X e Y de cada control determinan el orden de tabulación. Para especificar otro orden, establezca la propiedad **TabIndex** de uno o varios controles en un entero mayor que cero. Si, por ejemplo, el valor de un control se establece en 1, se activará antes que un control en el que el se establece en 2, o más. Para que [sea más fácil acceder](properties-accessiblity.md) a una aplicación, agregue información sobre herramientas y asegúrese de que la secuencia de tabulación de todas las pantallas es óptima.
5. **Personalizar el grosor de un control deslizante**.
   
    Para ajustan el valor de un [control deslizante](controls/control-slider.md), los usuarios arrastran su identificador a la derecha o la izquierda a lo largo de su carril.
6. **En dispositivos Android, pulse un botón para actualizar la lista de aplicaciones, en lugar de deslizar rápidamente el dedo hacia abajo**.
   
    Para más información, consulte [Uso de PowerApps en teléfonos o tabletas](run-app-client.md).
7. **Correcciones de errores y mejoras de rendimiento**.

1. **Abra aplicaciones en iOS 10**.

    Puede marcar una aplicación como favorita y que se muestren solo sus aplicaciones favoritas en la lista de aplicaciones que se pueden ejecutar.
3. **Deslice hacia abajo para actualizar la lista de aplicaciones en dispositivos iOS y Android**.
4. **Retrase la propiedad para los controles de entrada de texto**.
   
   Si establece la propiedad **[OnChange](https://powerapps.microsoft.com/en-us/tutorials/properties-core/)** para un control de **[entrada de texto](controls/control-text-input.md)** en una acción, esa acción se realizará, de forma predeterminada, cada vez que el usuario escriba un carácter en el control. Por ejemplo, un usuario puede escribir siete letras en un cuadro de búsqueda y los resultados se actualizarán cada vez que el usuario agregue una letra, lo cual consume innecesariamente ancho de banda de la red cuando se busca en un origen de datos. Si establece la propiedad **DelayOutput** para ese control en **true**, la acción no se produce hasta que el usuario escribe uno o más caracteres y, a continuación, deja de escribir en dicho control durante un período de tiempo.
5. **Se han agregado funciones matemáticas**.
   
   * Calcule el seno, el coseno, la tangente y otros valores trigonométricos.
   * Acceda fácilmente al valor de &pi;.
   * Convierta entre las unidades de ángulo.
   * Calcule logaritmos y potenciaciones naturales.
   
   Para más información, consulte [Funciones trigonométricas](functions/function-trig.md) y [Funciones numéricas](functions/function-numericals.md).

## <a name="release-20450"></a>Versión 2.0.450
1. **Agregue datos al personalizar un formulario**.
   
    Con un formulario seleccionado, pulse en el origen de datos para el formulario (o en **No se ha seleccionado ningún origen de datos**) en el panel derecho y, a continuación, pulse en **Agregar un origen de datos**.
2. **La creación de aplicaciones a partir de datos se ha delegado ya por completo para las conexiones de SQL Azure y Salesforce**.
   
   Al crear una aplicación a partir de datos de una conexión de SQL Azure o Salesforce, la aplicación resultante delegará ya todas las operaciones al servicio. Ya es posible trabajar con grandes conjuntos de datos con estas aplicaciones generadas.
   
   En lugar de generar fórmulas basadas en las funciones **Filter** y **Sort**, se generan ahora fórmulas basadas en las funciones **Search** y **SortByColumns**.  Estas nuevas funciones son más fáciles de delegar a un servicio de datos ya que no dependen de una fórmula arbitraria.
3. **Nueva función Buscar**.
   
   Actúa como una función de **filtro** especializada. La función **Search** toma una sola cadena para buscar y una lista de columnas de texto en las que realizar la búsqueda. Esta función devuelve también automáticamente todos los registros si no hay ningún término de búsqueda presente, lo cual resulta adecuado para su uso directo con un control de búsqueda de **entrada de texto**. Para más información, consulte la documentación sobre la función **Search**, o bien, eche un vistazo a las fórmulas que se han generado al crear una aplicación a partir de datos.
4. **Un operador ya admite la delegación**.
   
   No dude en usar este operador en la función **Filtrar** para lograr el mismo efecto que con la función **Buscar**.
5. **Muestre archivos PDF protegidos por contraseña**.
   
    Si configura un control de **PDF Viewer** para mostrar un archivo que requiere una contraseña, cada usuario deberá proporcionar la contraseña para ver el archivo. Para quitar el requisito de contraseña, establezca la propiedad **Password** de ese control en la contraseña de ese archivo.
6. **Hora universal coordinada en el control DatePicker**.
   
    De forma predeterminada, el control **DatePicker** interpreta todos los valores de fecha según la zona horaria local. Si la entrada se expresa como la medianoche de la hora universal coordinada (UTC), mostraría una fecha con un día menos en ciertas zonas horarias. Por ejemplo, un control **DatePicker** mostrará el 31 de diciembre de 2016 si establece su propiedad **DefaultDate** en "2017-01-01T00:00:00Z" y se encuentra en la zona horaria del Pacífico (UTC-0800). Para asegurarse de que el control muestra estos valores correctamente, cambie la propiedad **DateTimeZone** de **Local** a **UTC**.
